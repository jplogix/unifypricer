# Product Overview

Price Sync Dashboard is an automated product pricing synchronization system that connects StreetPricer API with e-commerce platforms (WooCommerce and Shopify).

## Core Functionality

- Fetches product prices from StreetPricer API
- Matches products across platforms using SKU identifiers
- Automatically updates prices in WooCommerce and Shopify stores
- Provides real-time dashboard for monitoring sync status
- Stores credentials securely using AES-256-GCM encryption

## Key Features

- Multi-store support (multiple WooCommerce/Shopify stores)
- Configurable sync intervals per store
- Product matching with confidence scoring
- Comprehensive error tracking and reporting
- SQLite database for local data persistence
