# Price Sync Connector Plugin - Installation Guide

The Price Sync Connector plugin allows you to connect your WooCommerce store to the Price Sync Dashboard with a single click.

## Features

- **One-Click Connection**: No manual API key entry required
- **Automatic Configuration**: Enables WooCommerce REST API and configures permalinks
- **Secure Key Generation**: Uses WooCommerce's native `WC_Auth` class to generate API keys
- **Step-by-Step Progress**: Visual feedback during connection process
- **Secure Transmission**: Credentials sent via encrypted POST (not URL parameters)

## Requirements

- WordPress 5.0 or higher
- WooCommerce 3.0 or higher
- Price Sync Dashboard account

## Installation

### Method 1: Manual Upload

1. Download the `price-sync-connector.zip` file
2. Go to your WordPress admin panel
3. Navigate to **Plugins → Add New**
4. Click **Upload Plugin**
5. Choose the zip file and click **Install Now**
6. Activate the plugin

### Method 2: FTP/SFTP

1. Extract the plugin zip file
2. Upload the `price-sync-connector` folder to `/wp-content/plugins/`
3. Go to **Plugins** in your WordPress admin
4. Find "Price Sync Connector" and click **Activate**

## Configuration

No manual configuration required! The plugin will:

1. Automatically enable WooCommerce REST API
2. Configure permalink settings if needed
3. Generate API keys securely when you connect
4. Send credentials to your Price Sync Dashboard

## Usage

1. **Install the Plugin**: Follow the installation steps above
2. **Go to Price Sync Dashboard**: Navigate to your dashboard
3. **Add Your Store**: Click "Add Store" and enter your store name and URL
4. **Click Connect**: The dashboard will open a popup to your WordPress admin
5. **Watch the Progress**: The plugin will automatically:
   - Check your WooCommerce installation
   - Enable the REST API
   - Configure permalinks
   - Generate API keys
   - Securely send credentials to the dashboard
6. **Done!** Your store is now connected and ready to sync

## What the Plugin Does

When you click "Connect Store" in your WordPress admin:

1. **Validates Environment** - Checks WooCommerce version and activation
2. **Enables REST API** - Ensures WooCommerce API is accessible
3. **Sets Permalinks** - Configures pretty permalinks if not set
4. **Generates API Keys** - Creates read/write keys using WooCommerce's native system
5. **Sends Credentials** - Securely POSTs encrypted credentials to Price Sync Dashboard
6. **Cleans Up** - Removes temporary data after successful connection

## Security

- API keys are generated using WooCommerce's built-in `WC_Auth` class
- Credentials are transmitted via encrypted POST requests (not in URL)
- Connection tokens expire after 15 minutes
- All sensitive data is cleared after successful connection
- No credentials are stored in logs

## Troubleshooting

### Plugin Page Shows "Token Required"

**Problem**: You accessed the plugin page directly from the WordPress admin menu.

**Solution**: Always connect from the Price Sync Dashboard. The plugin requires a valid connection token to work.

### Connection Fails at "Check WooCommerce"

**Problem**: WooCommerce is not installed or activated.

**Solution**:
1. Install WooCommerce from the WordPress plugin repository
2. Activate WooCommerce
3. Try connecting again

### Connection Fails at "Enable REST API"

**Problem**: Cannot enable WooCommerce REST API.

**Solution**:
1. Check that you have administrator permissions
2. Ensure WooCommerce is properly installed
3. Try deactivating and reactivating WooCommerce

### Connection Fails at "Generate API Keys"

**Problem**: Unable to create API keys.

**Solution**:
1. Ensure you're logged in as an administrator
2. Check that WooCommerce REST API is supported
3. Verify WooCommerce version is 3.0 or higher

### Connection Fails at "Send Credentials"

**Problem**: Cannot communicate with Price Sync Dashboard.

**Solution**:
1. Check your internet connection
2. Verify the Price Sync Dashboard backend is running
3. Check firewall/security settings aren't blocking outbound requests
4. Contact support if the issue persists

## Uninstallation

To remove the plugin:

1. Go to **Plugins** in WordPress admin
2. Deactivate "Price Sync Connector"
3. Click **Delete** to remove the plugin

**Note**: API keys generated by the plugin will remain in WooCommerce. To remove them:

1. Go to **WooCommerce → Settings → Advanced → REST API**
2. Find keys labeled "Price Sync Dashboard"
3. Click **Revoke** to delete them

## Permissions

The plugin requires:

- **Administrator role** - Only administrators can access the connection page
- **Network access** - Plugin must be able to communicate with Price Sync Dashboard backend
- **WooCommerce access** - Plugin needs to interact with WooCommerce API system
